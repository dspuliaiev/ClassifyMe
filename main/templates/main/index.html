About

Our project presents a full-stack web service specifically designed to classify images using advanced convolutional neural networks. The model was trained on the CIFAR-10 dataset, which allows the service to classify images into 10 distinct classes. The team employed a variety of technologies and tools to ensure a seamless and efficient user experience. Below, you'll find an overview of the key components and technologies that powered this project.

Front-end

We built a responsive and user-friendly interface that ensures a smooth interaction between users and the image classification service.

  • HTML/CSS
  • Bootstrap
  • JavaScript

Back-end

Our robust back-end efficiently handles data processing, model integration, and API management.

  • Python 3
  • Django

Utilites

We employed a set of powerful utilities to streamline development and deployment, maintaining code quality and collaboration.

  • VSCode / PyCharm
  • Git/GitHub
  • Docker

Neural Network

We utilized a range of powerful libraries and datasets to build, train, and deploy our convolutional neural network for image classification.

  • TensorFlow/Keras
  • Scikit-learn
  • NumPy
  • Pandas
  • CIFAR-10 Dataset

Our neural network efficiency and performance parameters

These parameters gauge neural network performance, encompassing validation accuracy and test accuracy.

Validation accuracy 88%
  • Total params: 3,091,594
  • Number of layers: 23
  • Number of convolutional layers: 6
  • The size of the kernel of the convolutional layers: 4
